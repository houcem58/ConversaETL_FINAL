#  ConversaETL

Conversational ETL Pipeline Generator with Multi-Agent System, LoRA Fine-Tuning, and RAG.

## Features
-  Natural language ETL generation
-  Multi-agent system (4 agents)
-  RAG-enhanced retrieval
-  LoRA fine-tuning
-  Auto-visualization
-  Gradio web UI

## Quick Start
\\\ash
pip install -r requirements.txt
export OPENAI_API_KEY=your-key
python examples/basic_example.py
python app.py  # Launch web UI
